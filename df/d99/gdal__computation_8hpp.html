Determines the neighboring coordinates of a given point in a 2D grid.

Generates a list of adjacent coordinates for a specified point within the confines of a grid defined by datasetHeight and datasetWidth. This function considers eight directions (N, NE, E, SE, S, SW, W, NW) surrounding the point, excluding out-of-bounds and the point itself.

Parameters
coordsThe coordinates of the point whose neighbors are to be identified.
datasetHeightThe height of the dataset in terms of number of points.
datasetWidthThe width of the dataset in terms of number of points.
Returns
Vector of Coords representing the neighboring points.
