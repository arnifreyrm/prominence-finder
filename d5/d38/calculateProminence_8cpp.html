Calculates peak prominences in a dataset using the water level method.

Processes a geographic dataset to determine the prominence of peaks. Peaks with prominence below the specified threshold are excluded from the output. The method simulates lowering water levels to identify and analyze individual islands (peaks).

Parameters
datasetUnique pointer to the GDALDataset being processed.
outputFilePathPath to the output CSV file for storing results.
prominenceThresholdMinimum prominence value for peaks to be included in the output.
verboseIf true, additional details like water level and active island count are printed during processing.
